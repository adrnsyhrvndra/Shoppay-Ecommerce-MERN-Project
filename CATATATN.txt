- Jika Error Oauth Playground "Akses diblokir: Permintaan aplikasi ini tidak valid"
1) Pergi Ke Google Console
2) Pilih Projectmu
3) Ke Menu Credentials
4) Tambah URL Authorized redirect URIs dengan : https://developers.google.com/oauthplayground
5) Authorize URL : https://mail.google.com/
6) Pilih Account Yang Sesuai
7) Pilih Lanjutkan
8) Nanti akan balik lagi ke halaman Oauth Playground & Authorization Code Akan Otomotasi Ter-generate
9) Klik Tombol Biru Exchange Authorization Code For Tokens
10) Akan Muncul Refresh Token dan Access Tokennya.
